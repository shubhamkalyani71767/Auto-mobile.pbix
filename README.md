

 
 
 🚗 Car Sales Dashboard (Power BI)

 📊 Project Overview

This project presents an interactive car sales dashboard built using Power BI. It provides insights into sales performance, pricing trends, and customer data to help understand business growth and key metrics.

---

 🎯 Key Features

* YTD (Year-To-Date) Total Sales Analysis
* Average Price Tracking
* Total Cars Sold Overview
* Monthly (MTD) performance comparison
* Dealer-wise and company-wise sales breakdown
* Interactive filters for:

  * Engine Type
  * Body Style
  * Dealer Name
  * Transmission

---

 📈 Dashboard Insights

* Total Sales reached **$176.9M** with a growth of **26.72%**
* Average Price observed at **$27.9K**
* Total Cars Sold: **6343 units**
* Sales distribution across multiple brands like Cadillac, Toyota, Audi, etc.
